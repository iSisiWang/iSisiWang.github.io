Hexo-Babel

2022/12/13 工具 Hexo 插件 Babel

Hexo-Babel 插件，使用 Babel 编译转换 JS 文件

安装

npm i hexo-babel -S

配置

babel:
    options:
    exclude:
        - "*.min.js"

options 详见 Options · Babel 和 @babel/preset-env · Babel

例如这是一种配置

babel:
    options: {
        "presets": [
            [
                "@babel/preset-env",
                {
                    "targets": "last 5 versions, not dead, > 0.3%"
                }
            ]
        ],
        "sourceType": "script"
    }
    exclude:
        - "*.min.js"

或者不填 options 参数，使用 babel.config.json

{
    "presets": [
        [
            "@babel/preset-env",
            {
                "targets": "last 5 versions, not dead, > 0.3%"
            }
        ]
    ],
    "sourceType": "script"
}
